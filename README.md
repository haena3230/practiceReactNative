### React Native 기반 Android Client 개발

### 소개
길고양이로 인한 도시 위생문제, 길고양이를 보살피는 지지자와 반대자들의 대립, 그로 인해 발생하는 혐오 범죄에 대한 해결책을 위한 프로젝트

### 제공기능
- 고양이 케어를 위한 기록 공간 제공
- 커뮤니티 공간 제공
- 현재 위치 기반 지도에 고양이 위치 기록
